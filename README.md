INTRODUÇÃO  

O projeto do site de viagens focado na Paraíba tem como objetivo criar uma plataforma digital intuitiva e informativa, destacando os principais destinos turísticos do estado. O sistema oferecerá uma experiência visual envolvente e acessível, garantindo que os visitantes possam navegar facilmente pelas seções que exploram praias, cultura, gastronomia e roteiros personalizados. O site será estruturado com uma página inicial que apresenta imagens impactantes dos cenários naturais da Paraíba, seguida por um menu de navegação organizado, permitindo acesso rápido às informações sobre pontos turísticos e dicas de viagem. Cada destino será apresentado com descrições detalhadas e elementos visuais que reforçam sua identidade, proporcionando ao usuário uma experiência imersiva sem necessidade de scripts complexos.
